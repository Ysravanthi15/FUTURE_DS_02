# 📊 Power BI Dashboard with Customer Churn Analysis

## 🚀 Project Overview  
This project presents an interactive Power BI dashboard developed during my internship.  
It focuses on analyzing data trends along with **Customer Churn Analysis** to understand customer behavior and retention patterns.

---

## 🎯 Objectives  
- Analyze key business metrics and trends 📈  
- Identify customer churn patterns and risks 🔍  
- Understand factors affecting customer retention 👥  
- Build interactive and user-friendly dashboards  
- Support data-driven decision-making  

---

## 📊 Dashboard Features  
- KPI indicators for quick insights 💡  
- Interactive visuals and charts  
- Customer churn analysis section  
- Trend analysis over time 📈  
- Segment/category-based comparisons  
- Dynamic filters and slicers  

---

## 🔍 Key Insights  
- Identified high churn rate segments and patterns  
- Recognized factors influencing customer retention  
- Highlighted areas requiring customer engagement improvement  
- Discovered trends that help reduce churn and improve loyalty  
- Provided actionable insights for business decisions
